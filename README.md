Lock Screen Overlay Android App
Описание
Приложение реализует экран блокировки с возможностью разблокировки свайпом. В приложении есть плавающая кнопка для вызова блокировки и кнопка для полного закрытия приложения.

Особенности:

Свайп зона с круговым индикатором разблокировки

Пульсирующая анимация и стрелка для подсказки действия

Запрос разрешения на отображение поверх других приложений

Плавающая кнопка для вызова блокировки

Как использовать
Установите приложение на Android устройство.

При первом запуске необходимо предоставить разрешение SYSTEM_ALERT_WINDOW.

Плавающая кнопка появляется на экране, нажмите её для показа экрана блокировки.

Разблокируйте экран свайпом круга вправо.

Для выхода используйте кнопку "Закрыть приложение".
